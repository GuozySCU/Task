今天来学习一下怎么用DiagrammeR画流程图吧 （后来在网上发现，似乎R还有一个diagram包也可以用来画流程图） 代码↓

#先画一个正常的足球比赛流程
library(DiagrammeR)
grViz("
digraph flowchart {
# node definitions with substituted label text
node [fontname = Helvetica, shape = rectangle] 
tab1 [label = '@@1']
tab2 [label = '@@2']
tab3 [label = '@@3']
tab4 [label = '@@4']
tab5 [label = '@@5']
# edge definitions with the node IDs
tab1 -> tab2 -> tab3 -> tab4 -> tab5;
}
[1]: '足球比赛'
[2]: '上半场'
[3]: '中场'
[4]: '下半场'
[5]: '结束'
")
